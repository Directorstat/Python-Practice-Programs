# Python-Practice-Programs

The program used to learn python

## Project Overview

This repository contains a collection of beginner Python programs. Each program demonstrates core programming concepts such as user input, conditional logic, loops, functions, lists, dictionaries, and exception handling.
The project focuses on simple problem-solving tasks often used in introductory programming courses and practice exercises.

## Programs Included

### 1. Statistics Quiz Game

A command-line quiz program that asks users basic statistics questions and awards points for correct answers.

Key features
* List based question system
* User input handling
* Score tracking
* Conditional statements

The topics covered for the questions
* Statistics definition
* Descriptive statistics
* Null hypothesis
* Dependent variables
* Basic inference concept

### 2. Number Swapping Program

This program swaps two numbers entered by the user using tuple unpacking.

Key concepts
* Input handling
* Tuple unpacking
* Exception handling for invalid input

Before Swapping:
First Number = 15.0
Second Number = 20.0

After Swapping:
First Number = 20.0
Second Number = 15.0

### 3. Number Identifier Program

This program checks whether a number entered by the user is:
* Prime number
* Armstrong number
* Palindrome number

Concepts used
* Functions
* Mathematical operations
* String reversal
* Conditional logic
e.g
You entered: 12
The number is neither Prime, Armstrong, nor a palindrome.

### 4. Fibonacci Sequence Generator

This program generates the Fibonacci sequence up to the number of terms specified by the user.

Concepts used

* Loops
* List storage
* Function creation

Example
Enter the number of Fibonacci terms to generate:  2
The first 2 terms of the Fibonacci sequence are:
[0, 1]

### 5. Python List and Dictionary Exercises

Short examples demonstrating common Python operations.

Topics covered

List pop operation
Removing the last element of a list

Dictionary keys extraction
Converting dictionary keys to a list

List comprehension
Creating a new list using conditional filtering

Loop control using break
Stopping a loop when a condition is met

List slicing
Selecting specific elements from a list

List concatenation
Combining two lists

Example snippets included

* Removing elements from lists
* Squaring even numbers using list comprehension
* Extracting dictionary keys
* Loop control statements
